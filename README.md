## Installation via go

### Clone git repo
```bash
git clone git@github.com:janritter/snipper.git
```

### Open project directory
```bash
cd snipper
```

### Install via go
```bash
go install
```

## Installation via Homebrew (For Mac / Linux)

### Get the formula
```bash
brew tap janritter/snipper https://github.com/janritter/snipper
```

### Install formula
```bash
brew install snipper
```

## Usage

### Run
```bash
snipper
```
